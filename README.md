# Faisal-Al-mosra-Final-Project

المشروع يساعد الطلاب المقبلين على الدراسه الجامعية بدون مايعرفون شيء عن الجامعة او طريقة الدراسة او التخصصات المتوفره او اذا كانت الجامعة حكوميه او اهليه وغيرها

الموقع هذا يساعد الطلاب بطريقة سهله عن طريق الطلاب المتخرجين او اللي ما زالوا يدرسون حيث يمكنهم من تقييم الجامعة والتخخصات والمعلمين وغيرها
